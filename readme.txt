Das Plugin erlaubt Dir den Spartacus Pride Kalender auf jeder beliebigen Seite deines Blogs einzubinden.

== Description ==
Der Spartacus International Gay Guide ist ein Reiseführer für schwule Männer. Mit dem Pride Kalender wird eine übersichtliche Liste von weltweiten Prides und Terminen angeboten (750+ Einträge).

= Was kann das Plugin =
Du kannst mit einem einfachen Shortcode ([pridecal_list]) in jede Seite und jeden Post den Spartacus Pride Kalender integrieren.

Die gesammte Funktionalität, inklusive Suche, Filter, usw. steht dann auf Deiner Seite zur Verfügung.

== Installation ==
Das Plugin wird ganz normal installiert und steht dann direkt zur Verfügung. Unter

Einstellungen > Mitfahrgelegenheiten

kannst Du alles anpassen.

== Frequently Asked Questions ==

= Die Farbe passt nicht zu meinem Wordpress Theme =
In den Einstellungen gibt es einen praktischen Color Picker, mit dem Du nach Belieben die Grundfarbe ändern kannst.

= Ich habe eine mehrsprachige Seite, kann ich das Plugin auf verschiedenen Seiten in unterschiedlichen Sprachen anzeigen? =
In den Einstellungen kannst Du einmal die Standardsprache festlegen. Aber in jedem Shortcode den Du einfügst, kannst Du den Parameter lang="xx" eintragen (xx steht hier für den Sprachcode, z.B. "de" für Deutsch) und die Ausgabe an dieser Stelle auf eine andere Sprache umstellen.
